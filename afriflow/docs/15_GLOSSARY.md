# 15 Glossary

> **Disclaimer**: Please read
> [DISCLAIMER.md](../DISCLAIMER.md). This is not a
> sanctioned project.

**AfriFlow** - The cross domain data integration
platform described in this repository.

**Golden ID** - A deterministic identifier that links
entities across domains.

**Domain Pod** - A country-level deployment that owns
local data and processing.

**Signal** - A derived insight produced by correlating
multiple domains.

**Schema Registry** - A system (e.g., Confluent Schema
Registry) that stores Avro schemas for Kafka topics.

**Medallion Architecture** - Data layering pattern with
Bronze (raw), Silver (cleaned), and Gold
(integrated) tables.

**Quality Score** - A numeric rating representing the
trustworthiness of a dataset.
